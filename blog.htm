Facebook Prophet Algorithm

Number of Covid-19 case prediction, Bangladesh · 11 June, 2020

Recently I was working with the Facebook Prophet algorithm. So as a side project I thought of predicting the total number of Covid-19 confirmed cases in Bangladesh for the next 60 days. The global dataset was taken from the "Datahub" website which is up-to-date and accurate. This algorithm is fairly simple to work with. If anyone is interested to learn then you can follow my Github repo.
