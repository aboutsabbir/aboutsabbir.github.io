My Achievements

Neural Networks and Deep Learning

This is the first course of the Deep Learning Specialization on Coursera. This course is conducted by Andrew NG. This is an introductory course of Neural Networks. Thanks to the course conductor and everyone at deeplearning.ai that I got the chance to learn a great deal about Machine Learning with python from this course.

Data Visualization with Python

This course is one of the courses of IBM Data Sceince Specialization. Almost all the plotting library of python is introduced in this course. Seaborn, matplotlib, pandas plotting library etc. Also there are a few extra graph plotting such as: word cloud, waffle chart is shown in this course. Overall this course is very productive.

Python Project: pillow, tesseract, and opencv

This course is conducted by Christopher Brooks, UM. This course introduces to the world of computer vision. How Python Image Library (PIL), OpenCV works is clearly explained in this course. Also how we can extract text from images easily using pytesseract with python is described in this course.

Convolutional Neural Networks

Course conductor the famous Andrew NG explained what Convolutional Neural Networks (CNN) is in this course. What CNN is and how is works is clearly explained in this course. And the course conductor also introduced different types of convolution networks such as: VGG16, AlexNet, LeNet5, ResNet etc.
